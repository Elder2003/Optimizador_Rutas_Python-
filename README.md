#Sistema de Rutas

Solución logística desarrollada con **Python** y **Flask** para calcular rutas de entrega eficientes. Utiliza algoritmos de geolocalización para resolver el problema del viajante (TSP) en tiempo real.

#Características
Optimización Inteligente:** Ordena automáticamente las paradas para minimizar la distancia.
Mapa Interactivo:** Interfaz oscura (Dark Mode) con marcadores personalizados usando **Leaflet.js**.
UX Moderna:** Panel flotante con notificaciones y diseño intuitivo.

Tecnologías
Backend: Python 3, Flask.
Frontend: JavaScript (Vanilla), Leaflet API.
Datos: OSRM (Open Source Routing Machine).

Cómo ejecutarlo
1. Clona el repositorio.
2. Instala dependencias: `pip install -r requirements.txt`
3. Ejecuta: `python run.py`
4. Abre el navegador en `http://localhost:5000`
